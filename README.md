## 📌 Project Overview

This project predicts whether a Flipkart product review is **Positive 😊** or **Negative 😡** using Machine Learning.

Users can enter product reviews and instantly get sentiment prediction through an interactive web application.

---

## ✨ Features

✔ Text Cleaning & Preprocessing  
✔ TF-IDF Vectorization  
✔ SVM Machine Learning Model  
✔ Streamlit Web Interface  
✔ Real-Time Sentiment Prediction  
✔ MLflow Experiment Tracking  
✔ AWS Deployment Ready  

---

## 🧠 Machine Learning Details

| Component | Technology |
|------------|-------------|
| Algorithm | Support Vector Machine (SVM) |
| Text Processing | TF-IDF |
| Evaluation Metric | F1 Score |
